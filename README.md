# WhatsApp Link Sender

## Overview

WhatsApp Link Sender is a simple web-based application that allows users to send a WhatsApp message to any phone number without saving it in contacts. The user enters a phone number and a message, and the app generates a WhatsApp link that opens directly in the browser or WhatsApp Web.

This project is built using basic HTML, CSS, and JavaScript, making it easy to understand, modify, and extend.

## Features

* Send WhatsApp messages without saving contacts
* Simple and clean user interface
* Uses WhatsApp official wa.me link format
* Lightweight and fast
* Easy to customize

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. Enter a valid WhatsApp phone number with country code (for example: 92xxxxxxxxxx).
2. Enter your message in the text area.
3. Click the "Send To" button.
4. A new tab will open with WhatsApp chat and your message pre-filled.

## Project Structure

* whatsapp-message-sender.html
  Contains the structure of the web page.
* styles.css
  Handles the design and layout of the application.
* script.js
  Contains JavaScript logic to generate and open the WhatsApp link.

## Setup Instructions

1. Clone or download this repository.
2. Open the project folder.
3. Open whatsapp-message-sender.html in any modern web browser.
4. Start sending WhatsApp messages.
